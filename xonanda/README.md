One-page static site: xonanda.uz

Files:
- index.html — main page
- styles.css — styling and animations
- script.js — tiny particles + parallax

How to view:
1) Open `index.html` in your browser directly (double-click or open file).

Or serve with a minimal static server (recommended):

```bash
# from the project folder
cd /home/manjaro/projects/xonanda
# Python 3
python -m http.server 8000
# then open http://localhost:8000
```
